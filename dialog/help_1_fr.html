Instructions : il existe quatre types de gestes tactiles, qui sont : toucher simple, toucher double, glisser et balayer. Ces gestes sont essentiels pour pleinement apprécier le jeu.
Mouvement
Vous pouvez déplacer votre personnage en touchant le sol ; ou en balayant l'écran pour faire apparaitre le pad.
Instructions du menu
1. Le menu et toutes les sous-interfaces sont déplaçables. Vous pouvez toucher puis glisser/déposer une fenêtre telle qu'un personnage, un sac ou de la magie où vous voulez.
2. Les éléments tels que la magie et les objets peuvent se placer sur vos boutons d'action. Il suffit de toucher un objet et de le glisser sur le bouton que vous voulez utiliser.
3. Vous pouvez toucher un objet, de l'équipement ou de la magie pour regarder leurs informations détaillées.
4. Pour pouvez toucher deux fois un objet pour l'utiliser. Par exemple, touchez deux fois pour boire une potion, s'équiper, ou lancer un sort.
5. À l'intérieur de votre sac, vous pouvez toucher et glisser/déposer un objet ou de l'équipement sur l'emplacement souhaité.

Boutons d'action
Instructions : il y a 16 boutons d'action au total. Le plus gros bouton est le bouton d'attaque. Autour du bouton d'attaque se trouvent 12 boutons vierges personnalisables. De plus, il existe 2 boutons de lancement sur soi et 1 bouton de changement de cible.
1. Bouton d'attaque : touchez une fois le bouton pour attaquer la cible une fois. Si la cible est amicale, touchez le bouton pour interagir.
2. Boutons vierges : vous pouvez glisser les objets, la magie ou l'équipement fréquemment utilisés sur ces boutons personnalisables. Les boutons personnalisables préfèrent viser votre cible actuelle.
3. Boutons de lancement sur soi : ces boutons fonctionnent de la même manière que les boutons vierges ci-dessus. La seule différence est que les boutons de lancement sur soi cibleront toujours le joueur lui-même.
4. Bouton de changement de cible : vous pouvez balayer vers la gauche pour changer de cible ennemie et vers la droite pour passer aux cibles amicales. Si vous balayez vers le haut, votre propre personnage sera sélectionné. Touchez le bouton pour sélectionner une cible aléatoire.
5. Maintien long : vous pouvez toucher et maintenir un bouton d'action pour le déplacer où vous voulez.
6. Balayez le bord supérieur du bouton d'attaque pour faire apparaître 12 boutons d'action vierges supplémentaires.
7. Boutons d'action de réinitialisation : vous pouvez restaurer les paramètres par défaut dans le menu du jeu.
