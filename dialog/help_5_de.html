Geltungsbereiche
1. Nicht-PVP-Zonen

Namensstatus
1. Rosa Name: Ein temporärer Status, der anzeigt, dass ein Spieler einen nicht-rosa oder roten Spieler angegriffen hat.
2. Roter Name: Dieser Status erscheint, wenn die Rechtmäßigkeitspunkte des Spielers bei -1000 oder darunter liegen.
3. Normaler Status: Blaue oder weiße Namen zeigen an, dass der Spieler im normalen Status ist.
4. Anzeigeprioritäten: Rot>Rosa>Normal

Strafen
1. Wenn man im normalen Status einen anderen normalen Spieler angreift, wird man für 90 Sekunden mit einem rosa Namen bestraft.
2. Wenn man im rosa Status einen normalen Spieler angreift, wird man für 90 Sekunden mit einem rosa Namen bestraft.
3. Wenn man im roten Status einen normalen Spieler angreift, wird man NICHT mit einem rosa Namen bestraft.
4. Wenn man in einem beliebigen Status einen roten oder rosa Spieler angreift, wird man NICHT mit einem rosa Namen bestraft.

Mordstrafen
1. Wenn man einen normalen Spieler tötet, wird man bestraft und verliert eine bestimmte Anzahl Rechtmäßigkeitspunkte.
2. Wenn man einen normalen Spieler tötet, werden die Wachen dich für 48 Stunden (Echtzeit) verfolgen.
3. Wenn man einen roten oder rosa Spieler tötet, erhält man keine Strafe.
