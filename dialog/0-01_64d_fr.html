Quête :
le cristal d'âme de belladone

Guide :
les belladones se trouvent dans le donjon de l'île promise. Gullveig vous attend sur l'île promise dans sa maison de la magie au sud.

Description :
la guérison inférieure est un sort magique de base qui peut guérir et purifier l'âme corrompue d'un monstre. Gullveig(32674,32830) vous demande de pratiquer la purification avec le sort de guérison inférieure sur le cristal d'âme d'une belladone. Récupérez le cristal d'âme purifié d'une belladone et rapportez-le à Gullveig.
