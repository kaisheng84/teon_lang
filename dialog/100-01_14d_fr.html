Quête :
utilisez le parchemin d'identification

Guide :
Patch se trouve sur l'(32759,32820) [île du ciel] au sud-est.

Description :
Patch vous a donné un parchemin d'identification pour vous aider avec le cadran solaire. Utilisez-le pour identifier vos gants en cuir de novice et révéler leurs capacités secrètes.
(Touchez le parchemin d'identification pour l'utiliser, puis touchez l'objet que vous voulez identifier.)
