Quête :
2 cœurs de Thri Kreen

Guide :
les Thri Kreen envoûtés se trouvent dans le donjon de Maros. Tiago est au nord-ouest de la ville de Maros.

Description :
Tiago(32839,33541) veut étudier comment les âmes mortes envoûtent les Thri Kreen dans le donjon de Maros. Pour ça, il a besoin que vous trouviez 2 cœurs de Thri Kreen, qu'il vous échangera contre la pierre du vide dont Karol a besoin.
